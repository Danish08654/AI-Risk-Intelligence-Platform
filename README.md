#  AI Risk Intelligence Platform

An end-to-end AI-powered risk analysis system that evaluates companies based on:

- Vendor Risk
- Financial Risk
- Compliance Risk
- Cyber Risk

Built using **FastAPI + Streamlit + Python agents architecture**

---

##  Features

- Real-time risk scoring engine
  
- Modular AI agent system
  
- Vendor & financial risk evaluation
  
- Compliance & cyber risk analysis
  
- Interactive Streamlit dashboard
  
- REST API backend (FastAPI)
  
- Docker support
